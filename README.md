# 🧠 AI Resume Generator (Python Project)

## 📌 Description
This project is a simple **AI Resume Generator** built using Python.  
It takes user input like name, education, skills, and projects, and generates a structured **ATS-friendly resume format**.

---

## 🚀 Features
- Takes user input dynamically
- Generates structured resume output
- ATS-friendly format
- Beginner-friendly Python project
- Uses basic string formatting and lists
